JS Based localtime that can be display in any page or post using [melt_localtime] shortcode.

== Description ==

## Use short code **[melt_localtime]** with attributes of

* text
* color
* size

Sample : **[melt_localtime text="Time now is " color="#e2cc1f" size="25px"]**
